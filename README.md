# 🤖空空bot
这只是一个简单的QQ框架

# 🎉使用方法
1.克隆项目
~~~sh
git clone https://github.com/huankong233/gugubot.git
~~~

2.安装[nodejs](https://nodejs.org/)和[mysql](https://www.mysql.com/)

3.导入数据库

4.安装支持库
使用
~~~sh
npm install -g yarn
yarn install
~~~

5.修改配置文件
~~~sh
nano config/bot.jsonc
~~~

6.运行框架
~~~sh
yarn test
~~~
